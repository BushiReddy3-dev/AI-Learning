# Topic 1: What is Deep Learning?

Deep Learning is a subset of Machine Learning that uses artificial neural networks with multiple hidden layers.

## Hierarchy

```text
AI
│
└── Machine Learning
      │
      └── Deep Learning
```

## Examples

- ChatGPT
- Microsoft Copilot
- GitHub Copilot
- Face Recognition
- Self Driving Cars

---

# Topic 2: What is a Neural Network?

A Neural Network is inspired by the human brain.

It consists of:

```text
Input Layer
      ↓
Hidden Layers
      ↓
Output Layer
```

## Example

```text
Employee Age
Experience
Salary
      ↓
Neural Network
      ↓
Will Employee Leave?
```

---

# Topic 3: Neural Network Components

## Input Layer

Receives data.

## Hidden Layer

Processes information.

## Output Layer

Produces prediction.

## Weights

Importance assigned to inputs.

## Bias

Adjusts predictions.

## Activation Function

Decides whether a neuron activates.

Examples:

- ReLU
- Sigmoid
- Softmax
