# Deep Learning Fundamentals

## What is Deep Learning?

Deep Learning is a subset of Machine Learning that uses Neural Networks.

## Neural Network Components

- Input Layer
- Hidden Layer
- Output Layer
- Weight
- Bias
- Activation Function

## CNN

Used for images.

Examples:

- Face Recognition
- Medical Imaging

## RNN

Used for sequential data.

Examples:

- Speech Recognition
- Translation

## Transformers

Used in modern AI systems.

Examples:

- GPT
- ChatGPT
- Copilot
- Claude
- Gemini

## Key Learning

Transformers are the foundation of Generative AI.
