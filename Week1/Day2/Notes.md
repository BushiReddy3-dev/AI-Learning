# Machine Learning Fundamentals

## What is Machine Learning?

Machine Learning is a subset of AI that enables systems to learn patterns from data and make predictions without explicit programming.

## Types of Machine Learning

### Supervised Learning

Uses labeled data.

Examples:
- Spam Detection
- House Price Prediction

### Unsupervised Learning

Uses unlabeled data.

Examples:
- Customer Segmentation

### Reinforcement Learning

Learns through reward and punishment.

Examples:
- Self Driving Vehicles

## ML Lifecycle

1. Data Collection
2. Data Cleaning
3. Feature Selection
4. Model Training
5. Testing
6. Deployment

## Key Learnings

- Features are inputs.
- Labels are outputs.
- Training data teaches the model.
- Testing data evaluates the model.
