The objective of this assignment is to understand the foundational concepts of Artificial Intelligence, Machine Learning, Deep Learning, Generative AI, and Large Language Models (LLMs).

---

## What I Learned

### 1. Artificial Intelligence (AI)

Artificial Intelligence is a branch of computer science focused on creating systems that can perform tasks that typically require human intelligence.

Examples:
- ChatGPT
- GitHub Copilot
- Siri
- Alexa
- Face Recognition Systems

---

### 2. Machine Learning (ML)

Machine Learning is a subset of AI that enables computers to learn patterns from historical data without being explicitly programmed.

Applications:
- Fraud Detection
- Recommendation Systems
- Predictive Analytics

---

### 3. Deep Learning

Deep Learning is a subset of Machine Learning that uses neural networks with multiple layers to solve complex problems.

Applications:
- Image Recognition
- Speech Recognition
- Natural Language Processing
- Large Language Models

---

### 4. Generative AI

Generative AI creates new content based on learned patterns from training data.

Generates:
- Text
- Images
- Code
- Audio
- Video

Examples:
- ChatGPT
- Microsoft Copilot
- Gemini
- Claude
- DALL-E

---

### 5. Large Language Models (LLMs)

LLMs are advanced deep learning models trained on massive amounts of text data.

Examples:
- GPT
- Gemini
- Claude
- Llama

Capabilities:
- Question Answering
- Content Generation
- Code Generation
- Summarization
- Translation

## AI vs Traditional Programming

### Traditional Programming

Input Data + Rules
↓
Program
↓
Output

### Machine Learning

Input Data + Expected Output
↓
Training
↓
Model

### Generative AI

Prompt
↓
Large Language Model
↓
Generated Output
