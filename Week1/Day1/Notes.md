# AI Fundamentals

## What is AI?

Artificial Intelligence enables machines to perform tasks requiring human intelligence.

## What is ML?

Machine Learning is a subset of AI where computers learn from data.

## What is Deep Learning?

Deep Learning uses neural networks with multiple layers.

## What is Generative AI?

Generative AI creates content such as text, images and code.

## What is LLM?

A Large Language Model is a deep learning model trained on massive amounts of text data.

Examples:
- GPT
- Gemini
- Claude
- Llama
