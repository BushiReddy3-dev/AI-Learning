# Part 1: Theory

## 1. What is Artificial Intelligence?

Artificial Intelligence is the field of computer science that focuses on building systems that can perform tasks that normally require human intelligence.

Examples:

- ChatGPT
- Microsoft Copilot
- GitHub Copilot
- Siri
- Alexa
- Face recognition
- Recommendation systems

### Simple Explanation

AI enables machines to understand, learn, reason, predict, generate content, and assist humans.

---

## 2. History of AI

### Important Milestones

| Year | Milestone |
|---|---|
| 1950 | Alan Turing proposed the question: Can machines think? |
| 1956 | The term Artificial Intelligence was introduced at the Dartmouth Conference |
| 1980s | Expert systems became popular |
| 1997 | IBM Deep Blue defeated chess champion Garry Kasparov |
| 2012 | Deep Learning became popular after breakthroughs in image recognition |
| 2017 | Transformer architecture was introduced |
| 2022 | ChatGPT made Generative AI popular worldwide |
| Today | AI Copilots, AI Agents, LLMs, and enterprise AI solutions are widely used |

---

## 3. Types of AI

### Narrow AI

Narrow AI is designed to perform a specific task.

Examples:

- ChatGPT
- Google Maps
- Netflix recommendations
- GitHub Copilot

### General AI

General AI means a system that can perform any intellectual task like a human.

Status: Not achieved yet.

### Super AI

Super AI means intelligence beyond human capability.

Status: Theoretical concept.

---

## 4. Machine Learning

Machine Learning is a subset of AI where systems learn patterns from data without being explicitly programmed.

### Example

Traditional programming:

```text
Rules + Data
     ↓
Program
     ↓
Output
```

Machine Learning:

```text
Data + Expected Output
        ↓
Training
        ↓
Model
        ↓
Prediction
```

Examples:

- Spam detection
- Fraud detection
- Loan approval prediction
- Product recommendations

---

## 5. Deep Learning

Deep Learning is a subset of Machine Learning that uses neural networks with multiple layers.

Used for:

- Image recognition
- Speech recognition
- Natural language processing
- Large Language Models
- Computer vision

Example:

```text
Image Input
    ↓
Neural Network
    ↓
Prediction
```

---

## 6. Generative AI

Generative AI creates new content such as text, images, code, audio, and video.

Examples:

- ChatGPT
- Microsoft Copilot
- GitHub Copilot
- DALL-E
- Gemini
- Claude

Generative AI can create:

- Emails
- Code
- Images
- Documents
- Summaries
- Reports
- Chatbot responses

---

## 7. Large Language Models

A Large Language Model is a deep learning model trained on large amounts of text data.

LLMs can:

- Understand prompts
- Generate text
- Summarize content
- Translate languages
- Generate code
- Answer questions

Examples:

- GPT
- Claude
- Gemini
- Llama

---

## 8. AI vs Traditional Programming

| Area | Traditional Programming | AI / ML |
|---|---|---|
| Logic | Written by developer | Learned from data |
| Input | Data and rules | Data and examples |
| Output | Rule-based output | Prediction or generated output |
| Best for | Fixed business rules | Pattern-based problems |
| Example | If-else validation | Fraud prediction |

5. Upload PNG and Draw.io files.

8. Click Commit.
9. Click Sync Changes or Push.
